📁 FileHelpect

Simple file downloader and bundler tool

About

FileHelpect lets you download individual files or generate a ZIP bundle of everything inside the files/ directory.

How to Download

  1. Clone this repository or download it as a ZIP:
    2. git clone https://github.com/your-username/FileHelpect.git
  2. Navigate into the folder:
    3. cd FileHelpect
  3. Run the Python script to generate a ZIP file:
    4. python generate_zip.py
  4. This will create downloadable_files.zip in the root directory, ready to be shared or extracted.
